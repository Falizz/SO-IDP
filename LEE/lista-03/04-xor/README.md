- Xor sem xor

Nesta atividade, você deverá realizar a operação lógica XOR entre os registradores
r8 e r13 e armazenar o resultado em rax.

Para esta atividade, as seguintes instruções NÃO estão disponíveis:

    - MOV, XCHG, XOR

Valores inicializados: - r8: 0x65232b2b1dd6a7e8 - r13: 0x9078d2d4e0dd2712

Obs: Lembre-se de não modificar os valores originais dos registradores citados.
