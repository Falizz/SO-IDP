- Desafio com instruções de operações lógicas bit a bit.

Utilizando apenas as instruções:

    - AND, OR, XOR

Implemente a seguinte lógica:

    if (x, y, w) possui exatamente um valor impar
        z = 1
    else:
        z = 0

onde:
x = r8
y = r9
w = r12
z = rax

Valores inicializados: - r8: 0x329c9981 - r9: 0x7a5f4ff - r12: 0x3005037d - rax: 0xffffffffffffffff

Restrições: - Somente as instruções AND, OR e XOR são permitidas. - Seu código deve ter no máximo 25 instruções.

Observação: serão executados múltiplos testes no código.
