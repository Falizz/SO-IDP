- Stack Stackando

Realize o seguinte:

- Retire os valores A e B do topo da pilha
- Execute a operação xor entre esses dois valores.
- Armazene de volta na pilha B R A, onde R é o resultado da operação xor.

Observação: a intenção é que os valores A e B voltem para a pilha, mas em ordem
invertida ao que estavam antes, com o resultado da operação xor entre eles.

Valores inicializados:

- (pilha) [0x7fffff1ffff0] = 0xf289355698bc1820 <- topo da pilha
- (pilha) [0x7fffff1ffff8] = 0xe2e13eee7254f007
