- Determinante

Nesta atividade, você deverá calcular o determinante da seguinte matriz 3x3:

        | a b c |
    D = | d e f |
        | g h i |

    a = r15
    b = r10
    c = rbx
    d = r13
    e = rdi
    f = r11
    g = rdx
    h = r14
    i = rcx

E o resultado deverá ser armazenado no registrador rax.

Valores inicializados:

- r15: 0x9e
- r10: 0x81
- rbx: 0x6a
- r13: 0x9d
- rdi: 0x7f
- r11: 0x78
- rdx: 0x34
- r14: 0x89
- rcx: 0x4a

Restrições (ou recomendações):

- Você não pode usar os registradores: rsi, r8, r9, r12
- A pilha está disponível para uso
