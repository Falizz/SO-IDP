 - Ler Memória -

Nesta atividade, você deverá escrever um código em assembly x86-64
para ler o conteúdo da memória contida no endereço 0x5544707f2000
Neste endereço consta o conteúdo do arquivo /desafio/recompensa.txt,
que é a resposta para a atividade.
Por último, você poderá utilizar somente as seguintes syscalls:
 - read
 - exit
Para esta atividade, o seu payload não poderá exceder 256 bytes.

Dica adicional: Todas as syscalls disponíveis são importantes, então
tente pensar em como utilizá-las.
