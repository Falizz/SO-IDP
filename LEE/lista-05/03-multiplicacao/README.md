- Multiplicação com Sinal

Nesta tarefa, você deverá realizar a múltiplicação com sinal entre duas qwords
(ambos > 0) que encontram-se no topo da pilha. O resultado deverá ser armazenado
no registrador RAX. Porém, caso o resultado da multilicação cause overflow, o
valor de RAX deverá ser -1.

Valores inicializados:

- (pilha) [0x7fffff1fffea] = 308349821042474122 <- Topo da pilha
- (pilha) [0x7fffff1fffe2] = 331

Observação: - Nesta atividade, serão executados múltiplos casos de testes.
