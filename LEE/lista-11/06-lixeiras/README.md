# Descrição

Nesta atividade, há três programas na pasta /desafio, que realizam uma ou
múltiplas alocações de memória. Em algum momento dessas manipulações, há
um ponteiro ptr que é alocado e liberado. O objetivo desta atividade é
responder, para cada programa, para qual bin a memória liberada de ptr,
está gerenciada, após a última alocação ou liberação realizada na main.
