## 2. Estrutura de Barramento

- Onde os dados são trafegados → **Linhas de Dados**  
- Controla o acesso → **Linhas de Controle**

## 3. Registradores x86-64

- Retorno de funções → **RAX**  
- Apontador de Instrução → **RIP**  
- Contadores → **RCX**  
- Registrador de Pilha → **RSP**  
- Base da Pilha → **RBP**  
- Propósito Geral → **R8**

- Controle industrial → **Tempo Real**
