- Leitura de Byte

Execute o seguinte:

- Leia um byte que encontra-se no endereço 0x404000 para o registrador r14
- Leia um byte que encontra-se no endereço 0x404001 para o registrador rbx
- Leia um byte que encontra-se no endereço 0x404002 para o registrador rsi
- Leia um byte que encontra-se no endereço 0x404003 para o registrador rdx

Valores inicializados: - [0x404000] = 0x5517531e
