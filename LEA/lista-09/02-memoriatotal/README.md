# Descrição

Nesta atividade, sua tarefa é verificar a quantidade total de memória RAM
disponível neste ambiente. Para isso, há diferentes formas de se obter essa
informação, como por exemplo consultando o arquivo /proc/meminfo, utilizando
ferramentas como free (caso estejam disponíveis) ou explorando outras interfaces
fornecidas pelo sistema operacional.

Para realizar a atividade, você deverá executar o programa `validador`, que irá
solicitar a resposta obtida, e caso esteja correta, você receberá o código de
resposta para a submissão da atividade.
